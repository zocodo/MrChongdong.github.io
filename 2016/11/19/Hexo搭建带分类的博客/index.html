Hexo搭建带分类的博客

  • 本文章基于使用 hexo 主题的博客，搭建的教程网上有很多，我就不再次写步骤了，推荐查看：https://xuanwo.org/2015/03/26/hexo-intor/
    注意：上面搭建步骤中的更换域名不是必须的，对域名没有要求的请忽略

  • 这一步解决对于 hexo 开启服务器无法检测文件更改从而刷新网页的

    • 全局安装 Browsersync
    • 安装结束后执行下面命令，能显示对应版本则表示安装成功
    • 安装 hexo 端 Browsersync 插件
    • 重启 hexo 服务器

  • 将博客的文章分类（如果主题不带menu项（步骤2），请换一个）

    1. 博客文件夹根目录下打开/scaffolds/post.md文件，在tags上添加categories:,文件会变成这样，以后每次执行 hexo n 文件名 都会带有categories项

    2. 打开你使用的主题文件夹，也就是 /themes/xxx，再用文件编辑器打开主题目录里的_config.yml文件，找到menu栏，你会看到类似于下面的那些配置

    3. 然后 cmd 执行hexo n 文件名创建一个新的文章，根据提示路径找到那个文件，编辑器打开后在categories:写上你的分类，例如categories: redux (这个就相当于路径categories/redux/)，也正好匹配了步骤2 中的 Redux 这个分类的路径

      然后在页面点击 Redux 按钮就可以看到设置为categories:redux所有文章，搭建一个带分类的blog就大功告成了
